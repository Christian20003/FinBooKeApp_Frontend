# FinBooKeApp - Frontend
This repository includes the frontend of the financial-book-keeping application. The backend is under the following link: https://github.com/Christian20003/FinBooKeApp_Backend. 

<big>IMPORTANT:</big> 
- This application is still in development and not ready for production.

# Goal
This application was designed to help individual users manage their finances. By recording income and expenditure and assigning them to different categories and payment methods, the aim is to make it as easy as possible for users to manage their household finances. Furthermore this is a student project.

# Instructions to run the frontend 

1. Clone this repository and and change to the directory:
    ```console
    cd ./FinBooKeApp_Frontend
    ```

2. Install all dependencies (ensure that npm is installed):
    ``` 
    npm install 
    ```

3. Start the server (in development mode):
    ``` 
    ng serve 
    ```
